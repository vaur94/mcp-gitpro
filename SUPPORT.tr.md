# Destek

English version: [SUPPORT.md](./SUPPORT.md)

## Nereden yardim istenir

- Bug ve tekrar uretilebilir urun problemleri: GitHub issue acin
- Ozellik onerileri ve kapsamli iyilestirmeler: GitHub issue acin
- Guvenlik hassasiyeti tasiyan bildirimler: `SECURITY.tr.md` veya `.github/SECURITY.md` icindeki ozel bildirim yolunu kullanin

## Neler eklenmeli

- Node.js ve npm surumleriniz
- Sunucuyu nasil baslattiginiz
- `MCP_GITPRO_GITHUB_TOKEN` degerinin tanimli olup olmadigi
- Sirlar temizlenmis `mcp-gitpro.config.json` ilgili bolumleri
- Sorun build veya test ile ilgiliyse `npm run ci:check` ciktilari

## Bu reponun desteklemedigi alanlar

Belgelenmis urun kapsami disindaki talepler genelde destek kapsami disinda kalir; buna yerel git otomasyonu, dosya sistemi duzenleme, shell calistirma, browser otomasyonu ve v1 icin HTTP transport dahildir.
