# Code of Conduct

Turkce surum: [CODE_OF_CONDUCT.tr.md](./CODE_OF_CONDUCT.tr.md)

This repository expects contributors and maintainers to communicate respectfully, review changes in good faith, and keep technical disagreement focused on the work rather than the person.

## Expected behavior

- Be respectful and specific in issues, pull requests, and reviews
- Assume good intent while asking for clarification or proposing changes
- Keep feedback technical, actionable, and repository-focused
- Avoid harassment, insults, discrimination, and personal attacks

## Scope

This code of conduct applies to repository discussions, issues, pull requests, review comments, and other project spaces managed through GitHub for `mcp-gitpro`.

## Reporting

If you experience or witness unacceptable behavior, ask the maintainers for a private reporting path before sharing sensitive details publicly.

This repository does not currently publish a dedicated private conduct mailbox or contact address.
